Support Vector Regression for Engineering Data

Introduction

This project explores how Support Vector Regression (SVR) can be used to model nonlinear relationships in engineering data. The goal was to build a regression model that predicts a continuous target variable from multiple input features more accurately than baseline linear models. Accurate prediction enables faster design iteration and reduces the need for repeated physical testing.

Methods

I cleaned and standardized the dataset, normalized numerical features, and implemented multiple SVR models using scikit-learn. I tested linear, polynomial, and RBF kernels. Hyperparameters (C, gamma, epsilon) were tuned via grid search with cross-validation. Baseline models included linear regression and ridge regression.

Results / Evaluation

The best model was the RBF-SVR, achieving significantly lower MSE and higher RÂ² than linear baselines. Residual analysis showed that the SVR captured nonlinear relationships that simple regression could not, especially at the extremes of the data range.
